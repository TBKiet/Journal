Revise the project scope:

This app is ONLY for me and my girlfriend.
It should run as a private web app in the browser, optimized for both laptop and iPhone.

Product requirements:
- Only 2 users: me and my girlfriend
- No public registration
- No complex couple-space system
- Use invite/password-based access or pre-created accounts
- Must feel like a cute, cozy, private digital diary for a couple

UI requirements:
- Extremely polished and cute UI
- Mobile-first for iPhone Safari
- Also beautiful on laptop browser
- Warm pastel color palette
- Rounded cards, soft shadows, subtle animations
- Cute empty states and microcopy
- Easy enough to use without explanation
- Bottom navigation on mobile
- Sidebar or top navigation on desktop
- Dark mode optional but nice to have

Core MVP:
1. Login
- Only two accounts allowed
- Example roles:
  - me
  - girlfriend
- No open signup
- Simple secure login

2. Home
- Days together counter
- Next anniversary countdown
- Latest memories
- Upcoming plans
- “Today’s question” card

3. Journal
- Write shared diary entries
- Add title, date, mood emoji, text, and photos
- Show entries as a cute timeline
- Filter by mood/tag/month

4. Photo memories
- Upload photos from iPhone or laptop
- Attach multiple photos to each diary entry
- Full-screen image preview
- Cute photo grid layout

5. Important dates
- First met
- First date
- Anniversary
- Birthdays
- Custom special days
- Count up/down display

6. Plans
- Add upcoming date ideas, trips, restaurants, movies, gifts, etc.
- Fields: title, date, note, location, status
- Mark done and optionally turn into a memory

7. Couple prompts
- Daily cute question
- Both can answer
- Show answers together

Tech stack:
- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Supabase Auth, Database, Storage
- Responsive design for iPhone and laptop

Important:
- Do not build for many couples.
- Do not over-engineer.
- Make the app feel personal, private, romantic, cute, and smooth.
- Prioritize beautiful UI and ease of use over complex features.

Design inspiration:
- A mix of Apple Journal, Between, Waffle, and a soft pastel Korean-style couple diary app.
- The interface should feel warm, intimate, minimal, and emotionally delightful.
